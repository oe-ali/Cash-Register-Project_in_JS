Cash Register

How To Use

Enter any amount of cash you want to give the cashier into the input field below.
The cash register itself can be used to set a price by using the numeric buttons.
The comma button can be used to display a price in decimals.
Make sure to click the ✔-Symbol after typing in a price total or else you won't proceed with the calculation.
You need to set the price.
To clear the price field click the ✘.
